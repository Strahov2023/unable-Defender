# Defender Control

Утилита для безопасного временного отключения защиты Microsoft Defender с автоматическим восстановлением, таймером и диагностикой. Версия 1.3.0.

## Запуск

1. Дважды щёлкните `DefenderControl.exe` — рекомендуемый запуск без консольного окна.
2. Подтвердите запрос контроля учётных записей Windows.
3. Выберите 5, 15, 30 или 60 минут и нажмите «Временно отключить».

`Start Defender Control.cmd` предназначен для диагностического запуска с видимым сообщением об ошибке.

Перед отключением создаётся одноразовое системное задание автоматического включения. Оно сработает после выбранного времени, даже если окно закрыто, пользователь вышел из учётной записи или компьютер в этот момент был выключен. Если создать и проверить задание не удаётся, защита не отключается.

В окне есть обратный отсчёт, автоматическое обновление состояния, отображение защиты от подделки и кнопка немедленного включения. Одновременный запуск двух копий заблокирован. Программа умеет сворачиваться в системный трей, показывает уведомления и копирует техническую диагностику.

Утилита также показывает состояние «Интеллектуального управления приложениями» (Smart App Control) и открывает соответствующий раздел «Безопасности Windows». Из-за возможной необратимости этого действия отключение подтверждается вручную в системном интерфейсе Windows.

Журнал работы хранится в `%LOCALAPPDATA%\DefenderControl\DefenderControl.log` и открывается кнопкой из приложения.

## Проверка безопасности выпуска

`Test-Release.ps1` проверяет синтаксис, привязку EXE к SHA-256 скрипта, обнаружение намеренной подмены, защитные ACL/reparse-point барьеры, манифест и опасные шаблоны. Для финального подписанного выпуска используйте ключ `-RequireSignature`.

Утилита не отключает Tamper Protection, службы Defender, обновления или брандмауэр. Если Windows блокирует изменение, управляйте разрешёнными исключениями через приложение «Безопасность Windows».
